## 2. GitHub Action Secrets
To configure automatic signing in GitHub Actions, go to your GitHub repository -> **Settings** -> **Secrets and variables** -> **Actions** -> **New repository secret** and add the following:

- **KEYSTORE_BASE64**: (Copy the Base64 string provided in the final response)
- **KEY_ALIAS**: `arfix`
- **KEY_PASSWORD**: `arfix123`
- **STORE_PASSWORD**: `arfix123`
